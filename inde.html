C Programming Help

Basic Syntax

C programs start with #include <stdio.h> for input/output operations. The main() function is the entry point of every C program.

Variables

Declare variables with their type: int age = 25; float price = 99.99; char grade = 'A';

Printf Function

Use printf() to display output: printf("Hello %s, you are %d years old", name, age);

Loops

for(int i=0; i<10; i++) { /* code */ } creates a loop that runs 10 times.
